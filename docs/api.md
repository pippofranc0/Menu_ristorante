# API utilizzate

Questa pagina descrive le API utilizzate dal progetto **Menu Ristorante**.


## API locale
In questo esercizio l'API è simulata dal file locale:

```text
data.json
```

Il file data.json contiene piatti con prezzo, categoria, allergeni e disponibilità.

Il file script.js carica i dati con:

```text
fetch("data.json")
```

Quindi il progetto deve essere aperto tramite un piccolo server locale, non solo con doppio click sul file HTML.


## Note

```text
data.json
```

fornisce dati fittizi. Questo significa che:

- i menu non sono reali
- la lista allergeni non è affidabile
- i dati possono essere usati liberamente per test e progetti didattici
- non è necessario configurare un database
